HTTP/1.1 — Backbone of the Early Web

HTTP/1.1 stabilized semantics and performance for the early modern web.

Highlights

  • Persistent connections and pipelining reduce connection churn.
  • Host header enables virtual hosting on shared IPs.
  • Chunked transfer streams unknown-length responses.
  • Cache directives improve freshness and efficiency.
  • Content negotiation supports language and format variants.

Limitations

TCP head-of-line blocking and serialized behavior under loss kept latency high for asset-heavy pages.
